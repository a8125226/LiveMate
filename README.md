# プログラムのタイトル

LiveMate

# プログラムの概要

LiveMateは、ライブやコンサート、フェスなどに参加する人を対象としたWebアプリです。公演情報、持ち物、予算、グッズ、タイムスケジュール、会場情報、移動方法、セットリスト、ライブ日記など、ライブに関する情報を一つのアプリで管理できます。ライブ前の準備から当日、終了後の思い出の保管までをサポートし、利用者がより充実したライブ体験を楽しめるようにすることを目的としています。

# プログラムの実行方法

1. 「index.html」または公開URL（https://a8125226.github.io/LiveMate/）を開く。
2. 「新規ライブ追加」から、公演名、アーティスト名、会場、日付、開場時間、開演時間などを入力し、ライブ情報を登録する。
3. 登録したライブを選択し、持ち物チェック、予算管理、グッズ管理、タイムスケジュール、会場情報などを入力する。
4. 最寄駅や会場情報を入力することでアクセス方法や近くのご飯屋さんなどをマップを通して検索する。
5. ライブ終了後は、セットリストやライブ日記、写真などを記録する。
6. 入力した情報はブラウザ内に保存され、同じ端末・ブラウザで再度開いた際にも確認できる。
